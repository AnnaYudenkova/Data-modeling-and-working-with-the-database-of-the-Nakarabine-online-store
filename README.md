# Моделирование данных и работа с базой данных интернет-магазина «Накарабине»
Данный проект представляет собой доработку и завершение первоначальной версии спецификации требований для веб-приложения интернет-магазина «Накарабине». В рамках задачи была проделана критически важная работа по созданию недостающих артефактов — логической модели данных и словаря данных, — без которых была невозможна дальнейшая разработка продукта (создание базы данных и backend-логики).
